### Introduction

SkillsHub is a unified Agent Skills management platform that allows you to use the same set of skills across multiple Agent tools (Claude Code, Cursor, Gemini CLI, OpenCode, etc.).

### âœ¨ Key Features

- ğŸ¯ **Unified Management** - Install once, sync to multiple tools

- ğŸ§° **Custom Tools** - Add custom AI coding tools and paths
- ğŸŒ **Multilingual UI** - Built-in 9 languages

